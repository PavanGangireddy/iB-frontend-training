# 1.6 Arrow Functions

* With arrow functions, we are able to omit the curly braces and the return statement due to implicit returns
* Example 1:

```javascript
function timesTwo(params) {
  return params * 2
}
timesTwo(4);  // 8

// Using Arrow functions

var timesTwo = params => params * 2
timesTwo(4);  // 8
```
